# Mission Control Web

AI Agent Control Center - OpenClaw Dashboard

## 🚀 在线访问

https://jack.github.io/mission-control-web

## 🛠️ 本地开发

```bash
npm install
npm run dev
```

## 📦 构建

```bash
npm run build
```

## 🎨 功能

- Dashboard - 主控制台
- Agents - Agent管理
- Memory - 记忆系统
- Tasks - 任务管理
- Analytics - 数据分析
